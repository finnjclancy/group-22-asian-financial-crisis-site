# Group 22 Asian Financial Crisis Site

Interactive website companion for a report on the 1997-1998 Asian Financial Crisis.

## Files

- `index.html` - main site
- `styles.css` - visual design and layout
- `script.js` - interactive charts and scroll behavior
- `favicon.svg` - tab icon
- `Report.docx` - source report used for the site
- `asian_crisis_sources.md` - source log

## Run locally

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.
